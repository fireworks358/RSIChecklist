# Emergency Airway Guidelines - PDF Files

This folder contains the clinical guideline PDF files that will be displayed in the Emergency Airway Portal.

## Required PDF Files

### Adult Guidelines (`adult/` folder)
1. `rsi-checklist.pdf` - Rapid Sequence Intubation Checklist for Adult Patients
2. `difficult-airway-algorithm.pdf` - Management Algorithm for Difficult Adult Airways
3. `cico-algorithm.pdf` - Can't Intubate, Can't Oxygenate Emergency Protocol
4. `failed-intubation-drill.pdf` - Step-by-Step Protocol for Failed Intubation Scenarios

### Paediatric Guidelines (`paediatric/` folder)
1. `rsi-checklist.pdf` - Rapid Sequence Intubation Checklist for Paediatric Patients
2. `difficult-airway-algorithm.pdf` - Management Algorithm for Difficult Paediatric Airways
3. `cico-algorithm.pdf` - Can't Intubate, Can't Oxygenate Emergency Protocol
4. `failed-intubation-drill.pdf` - Step-by-Step Protocol for Failed Intubation Scenarios

## Adding Your PDF Files

1. Place your clinical guideline PDF files in the appropriate folders (`adult/` or `paediatric/`)
2. Ensure the filenames match exactly as listed above
3. The service worker will automatically cache these files for offline access
4. Maximum file size: 10MB per PDF

Currently, placeholder PDF files have been created. Replace these with your actual clinical guidelines before deploying the application.
